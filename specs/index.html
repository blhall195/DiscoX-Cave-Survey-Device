DiscoX Device Specifications

These specifications are based on the SAP platform, the DiscoX uses identical sensor modules.

Specifications

  • 100m range (in ideal conditions)
  • Bluetooth enabled, connects to TopoDroid and SexyTopo
  • Intuative 4 button interface
  • Gorrilla Glass display cover
  • Quarz Glass display cover
  • Powerful M4 processor for better stability
  • Configurable settings
  • Reprogrammable
  • Waterproof USB-C port
  • RGB LED indicator light
  • Replacable parts
  • Built in hole for lanyard
  • Very waterproof design - needs testing
  • 1200mAh battery, average current draw 125mA
